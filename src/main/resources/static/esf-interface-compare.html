💡 使用说明

  • Excel结构要求：
    • 每个sheet页代表一个交易码
    • 每个sheet页包含"输入"和"输出"两个部分
    • 通过A列的"输入"、"输出"来识别各部分
    • 字段列：A列（英文名称）、B列（中文名称）、C列（数据类型）、D列（是否必输）
  • 比对规则：
    • 只比对A、B、C、D四列（英文名、中文名、数据类型、是否必输）
    • 以A列的英文名称作为唯一值进行匹配
    • 输入部分和输出部分分别比对
  • 修订记录格式：
    • A列：序号（从1开始递增）
    • B列：交易码（sheet页名称）
    • C列：新增/删除/修改（带颜色标记）
    • D列：修订内容（输入和输出的所有差异，单元格内换行展示）
  • 修订内容示例：
    • 字段删除：opertInsID字段 删除
    • 字段新增：ostID字段 新增
    • 字段修改：cstAcNum字段 【中文名称】：子账户序号 --> 账户序号 【数据类型】：CHAR(10) --> CHAR(20) 【是否必输】：N --> Y
