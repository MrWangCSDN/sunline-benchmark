💡 使用说明

  • 文件说明：
    • 左边：XML表定义文档（Excel格式）
    • 右边：迁移中间表文档（Excel格式）
    • XML表定义中的"表总览"sheet页自动跳过，不参与比对
  • Sheet页比对规则：
    • 左边sheet多：标记为"表新增"（绿色）
    • 右边sheet多：忽略，不做任何处理
    • Sheet名称相同：进行字段级别比对
  • 字段识别：
    • XML表定义：从"字段名"行开始，到"索引ID"行结束
    • 迁移中间表：从第11行开始
    • 以字段名（XML的A列，迁移中间表的B列）作为唯一标识
  • 字段映射规则：
    • XML A列（字段名） → 结果 B列
    • XML B列（中文名） → 结果 C列
    • XML C列（数据库类型） → 结果 D列
    • XML D列（长度） → 结果 E列
    • XML F列（空值） → 结果 F列
    • XML H列（默认值） → 结果 H列
    • 结果 A列：从1开始递增的序号
    • 结果 G/I/J/K/L列：从迁移中间表复制
    • 结果 M列往后：从迁移中间表复制
  • 字段变更标记：
    • ✅ 字段新增（左有右无）：绿色背景显示
    • ✏️ 字段修改：黄色背景显示最新值（不显示旧值→新值）
    • ❌ 字段删除（左无右有）：灰色背景+删除线，显示在sheet页最下面
  • 修订记录格式：
    • A列：表名
    • B列：修订级别（表/字段）
    • C列：修订方式（新增/删除/修改，带颜色标记）
    • D列：修订明细（带双向超链接）
  • 超链接：
    • 修订记录 → 目标表字段行（正向链接）
    • 表字段行 → 修订记录（反向链接）
