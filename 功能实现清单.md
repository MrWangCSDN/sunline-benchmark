## 功能特性

### 核心功能

1. **多线程扫描**
   - 每个jar包独立线程处理
   - 线程池大小：5个线程
   - 提高扫描效率

2. **智能匹配**
   - 自动忽略jar包版本号
   - 只匹配jar包名称前缀
   - 支持模糊匹配

3. **实时进度**
   - 实时显示扫描进度百分比
   - 显示每个jar包的扫描状态
   - 统计扫描结果（成功/失败数量）

4. **XML解析**
   - 解析flowtrans.xml文件
   - 提取交易基本信息
   - 提取流程步骤信息（service/method）
   - 自动记录步骤顺序

5. **数据管理**
   - 自动更新已存在的交易
   - 删除旧的流程步骤
   - 保证数据一致性

6. **用户交互**
   - 支持全选/取消全选jar包
   - 支持取消扫描操作
   - 友好的错误提示
   - 美观的UI界面

### 可扫描的Jar包

- ap-tran（应用交易）
- dept-pbf（部门业务）
- loan-pbf（贷款业务）
- sett-pbf（结算业务）
- comm-pbf（通用业务）

## 使用流程

1. **初始化数据库**
   ```bash
   mysql -u<用户名> -p<密码> <数据库名> < src/main/resources/sql/create_jar_scan_tables.sql
   mysql -u<用户名> -p<密码> <数据库名> < src/main/resources/sql/add_jar_scan_menu.sql
   ```

2. **启动应用**
   ```bash
   mvn spring-boot:run
   ```

3. **访问功能**
   - 登录系统
   - 导航到"Git代码管理" -> "扫描分析"

4. **执行扫描**
   - 输入Fat Jar的lib目录路径
   - 选择要扫描的jar包
   - 点击"开始扫描"

5. **查看结果**
   - 实时查看扫描进度
   - 查看每个jar包的扫描结果
   - 查看数据库中的交易和步骤信息
