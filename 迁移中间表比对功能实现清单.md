## 核心功能特性

### 1. Sheet页比对规则
- ✅ 跳过所有包含"表总览"字样的sheet（xxx表总览、表总览等）
- ✅ 左边sheet多：标记为"表新增"
- ✅ 右边sheet多：忽略不处理
- ✅ Sheet名称相同：进行字段级别比对

### 2. 字段映射规则
```
XML A列（字段名） → 结果 B列
XML B列（中文名） → 结果 C列
XML C列（数据库类型） → 结果 D列
XML D列（长度） → 结果 E列
XML F列（空值） → 结果 F列
XML H列（默认值） → 结果 H列
结果 A列：从1开始递增
结果 G/I/J/K/L列：从迁移中间表复制
结果 M列往后：从迁移中间表复制
```

### 3. 字段变更标记
- ✅ **字段新增**（左有右无）：绿色背景显示
- ✅ **字段修改**：黄色背景显示最新值（**不显示旧值→新值**）
- ✅ **字段删除**（左无右有）：灰色背景+删除线，显示在sheet页最下面
- ✅ **字段未变更**：白色背景

### 4. 修订记录
- ✅ 放在第一页
- ✅ 格式：A列-表名、B列-修订级别、C列-修订方式、D列-修订明细
- ✅ 新增-绿色、删除-灰色、修改-黄色
- ✅ 包含双向超链接（修订记录 ↔ 目标表字段行）

### 5. 其他特性
- ✅ 多线程并行处理，提高性能
- ✅ 复用ExcelCompareServiceImpl代码
- ✅ 独立的MIGRATION_TABLE模式，不影响其他功能
- ✅ 支持.xlsx和.xls格式
- ✅ Zip bomb检测阈值调整（支持高度压缩的文件）

## 使用流程

1. **执行菜单SQL**
   ```bash
   mysql -u用户名 -p数据库名 < src/main/resources/sql/add_migration_table_compare_menu.sql
   ```

2. **重启应用**
   ```bash
   ./restart.sh
   ```

3. **访问功能**
   - 登录系统
   - 工具箱 -> 🔄 迁移中间表比对
   - 上传XML表定义文档（左）和迁移中间表文档（右）
   - 点击"开始比较"
   - 查看统计结果
   - 下载比对结果文件

## 代码复用说明

本功能基于现有的Excel比对框架实现：
- ✅ 复用 `ExcelCompareServiceImpl` 核心引擎
- ✅ 复用 `StyleCache` 样式缓存机制
- ✅ 复用 `RevisionRecord` 修订记录结构
- ✅ 复用多线程并行处理框架
- ✅ 独立的比对模式，通过 `CompareMode.MIGRATION_TABLE` 区分
- ✅ 不影响现有的Excel比对、XML-DB比对、ESF接口比对功能

## 技术栈

- Spring Boot 3.1.1
- Apache POI 5.2.3（Excel处理）
- 多线程并发（ExecutorService）
- RESTful API

## 注意事项

1. XML表定义中所有包含"表总览"字样的sheet都会被跳过
2. 字段删除时，会在结果sheet的最下面显示（灰色+删除线）
3. 字段修改时，只显示最新值，不显示"旧值→新值"
4. 只处理XML表定义中存在的表，迁移中间表中多出的表会被忽略
5. 字段以字段名作为唯一标识进行匹配
