# 迁移中间表比对功能说明

## 功能概述

迁移中间表比对功能用于比较XML表定义文档和迁移中间表文档的差异，帮助识别字段的新增、删除和修改。

## 使用方法

### 1. 访问功能
- 登录系统后，点击左侧菜单"工具箱"
- 选择"🔄 迁移中间表比对"子菜单

### 2. 上传文件
- **左边**：上传XML表定义文档（Excel格式）
- **右边**：上传迁移中间表文档（Excel格式）

### 3. 开始比对
- 点击"开始比较"按钮
- 系统将自动比对两个文档并生成结果
- 比对完成后，显示统计信息并提供下载链接

## 比对规则

### Sheet页处理规则
1. **跳过"表总览"**：XML表定义中所有包含"表总览"字样的sheet都不参与比对（例如：表总览、xxxx表总览等）
2. **左边多**：如果XML表定义中有某个表，但迁移中间表中没有，标记为"表新增"
3. **右边多**：如果迁移中间表中有某个表，但XML表定义中没有，**忽略不处理**
4. **相同表名**：两边都存在的表，进行字段级别比对

### 字段识别规则
- **XML表定义**：字段部分从"字段名"行开始，到"索引ID"行结束
- **迁移中间表**：字段部分从第11行开始

### 字段映射规则

| XML列 | 列名 | 迁移中间表列 | 结果列 | 说明 |
|-------|------|-------------|-------|------|
| A | 字段名 | B | B | 字段英文名 |
| B | 中文名 | C | C | 字段中文名 |
| C | 数据库类型 | D | D | 数据类型 |
| D | 长度 | E | E | 字段长度 |
| F | 空值 | F | F | 是否允许空值 |
| H | 默认值 | H | H | 默认值 |
| - | **序号** | - | **A** | **从1开始递增** |
| - | G/I/J/K/L | G/I/J/K/L | G/I/J/K/L | 从迁移中间表复制 |
| - | M列往后 | M列往后 | M列往后 | 从迁移中间表复制 |

### 字段变更标记

1. **字段新增**（左有右无）
   - 显示方式：绿色背景
   - 修订记录：标记为"新增"，绿色显示
   - 修订内容：`[字段名] 字段新增`

2. **字段修改**（两边都有，值不同）
   - 显示方式：黄色背景，**只显示最新值**（不显示旧值->新值）
   - 修订记录：标记为"修改"，黄色显示
   - 修订内容：`[字段名] 字段修改`

3. **字段删除**（左无右有）
   - 显示方式：在结果sheet中显示在最下面，灰色背景+删除线
   - 修订记录：标记为"删除"，灰色显示
   - 修订内容：`[字段名] 字段删除`
   - 包含双向超链接

4. **字段未变更**
   - 显示方式：白色背景，正常显示

## 结果文件

### Sheet结构
1. **修订记录**（第一页）
   - A列：表名
   - B列：修订级别（表/字段）
   - C列：修订方式（新增/删除/修改，带颜色）
   - D列：修订明细（带双向超链接）

2. **各表Sheet**（按XML顺序）
   - 包含所有匹配成功的表
   - 字段按照映射规则重新排列
   - A列从1开始递增
   - 变更字段标记颜色

### 双向超链接
- **正向**：从修订记录 -> 目标表的具体字段行
- **反向**：从表字段行 -> 修订记录

## 统计信息
- ✅ 新增表数量
- ✏️ 修改表数量
- 📝 总计表数量

## 注意事项
1. 只处理XML表定义中存在的表，忽略迁移中间表中多出的表
2. 字段修改时，结果中只显示最新值，不显示"旧值->新值"的变化
3. 字段删除只在修订记录中体现，不在结果sheet中显示删除的行
4. M列往后的所有列都从迁移中间表中复制
5. 文件格式支持：`.xlsx` 和 `.xls`

## 技术实现
- 基于 `ExcelCompareServiceImpl` 复用代码
- 使用 `MIGRATION_TABLE` 比对模式
- 支持多线程并行处理，提高性能
- 使用Apache POI处理Excel文件
